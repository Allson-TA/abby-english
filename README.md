# Abby 單字練習

給英文初學者（CEFR A1–A2）的單字練習網站。單一 HTML 檔，開啟即用，不需安裝。

👉 **網址：** https://allson-ta.github.io/abby-english/

## 有什麼

| 模式 | 說明 |
|------|------|
| ✏️ 填空遊戲 | 看句子選正確單字，換題會複誦單字與整句 |
| ✅ 對錯判斷 | 判斷句子敘述對不對，可手動聽整句 |
| 🔤 拼字挑戰 | 看中文拼出英文，可用提示 |
| 🃏 配對遊戲 | 英文配中文 |
| 👂 聽力・單字 | 聽發音選出正確單字 |
| 🎧 聽力・句子填空 | 聽整句選出空格的單字 |
| 🎤 口說練習 | 念出英文，系統辨識並評分 |

- **答錯會停下來**讓你看完解析，按「下一題」才繼續；答對則自動換題
- **自適應出題**：常錯的字出現頻率提高，已熟練的字降低
- **可自行新增單字**：一次一個或一次多個，有防呆檢查
- 學習紀錄存在自己的裝置，可匯出／匯入 JSON 搬移

## 手機使用

用 Safari（iPhone）或 Chrome（Android）開上面的網址即可。
建議按「分享 → 加入主畫面」，之後像 App 一樣點開就能用。

## 注意事項

- **口說練習**需要允許麥克風權限；iPhone 需 iOS 14.5 以上的 Safari。
- 學習紀錄存在**各自的瀏覽器**裡，換裝置不會自動同步 —— 請用「學習統計 → 匯出／匯入」搬移。
- 清除瀏覽器資料會一併清掉學習紀錄，重要進度請先匯出備份。
